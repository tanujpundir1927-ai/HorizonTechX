# HorizonTechX - Python Mini Projects

Ye repository 4 basic Python projects ka collection hai jo core programming concepts jaise loops, conditionals, dictionaries, file handling, aur automation ko demonstrate karte hain.

## 📂 Projects List

### 1. Hangman Game (`task1_hangman.py`)
Ek text-based Hangman game jahan player ek predefined list mese chune gaye word ko letter-by-letter guess karta hai. Player ko 6 galat guesses ki limit di gayi hai.

**Concepts Used:** `random`, `while loop`, `if-else`, `strings`, `lists`

### 2. Stock Portfolio Tracker (`task2_stock_tracker.py`)
User apne stocks (jaise AAPL, TSLA) aur unki quantity input karta hai, aur script hardcoded prices ke basis par total investment value calculate karke ek `.txt` file mein save kar deti hai.

**Concepts Used:** `dictionary`, `input/output`, `basic arithmetic`, `file handling`

### 3. Email Extractor (`task3_email_extractor.py`)
Ye script ek `.txt` file ko read karke usmese saari email addresses ko `regex` (regular expressions) ki madad se extract karti hai aur unhe ek nayi file mein save kar deti hai.

**Concepts Used:** `re`, `file handling`, `os`

### 4. Basic Rule-Based Chatbot (`task4_chatbot.py`)
Ek simple chatbot jo user ke basic greetings jaise "hello", "how are you", aur "bye" ka predefined tareeke se jawab deta hai.

**Concepts Used:** `if-elif`, `functions`, `loops`, `input/output`

## 🚀 How to Run
Kisi bhi file ko run karne ke liye terminal mein niche di gayi command use karein:

\`\`\`bash
python task1_hangman.py
\`\`\`
